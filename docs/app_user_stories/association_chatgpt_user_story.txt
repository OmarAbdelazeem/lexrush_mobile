This game is a semantic association / word meaning matching game. The player sees a target word at the top and two candidate words at the bottom. They must tap the word that has the closest meaning or association to the target word.

It is different from Antonym Rush and Agility because it is more about word relationship recognition than movement or fast object tracking.

Game: Association
Core idea

The player sees a target word and must choose which of two words is most closely associated with it.

Example from the video:

When the player chooses correctly, the selected node glows and the game advances. When wrong, the selected answer is marked incorrect, and the game shows a short explanation.

Complete User Story

As a player, I want to quickly identify which word is most closely associated with the target word, so that I can improve vocabulary, semantic reasoning, and fast word association skills.

Gameplay Summary

Each round shows:

one target word at the top
two possible associated words at the bottom
a simple branching line/graph connecting the target to the two options
a timer
score
feedback after correct/wrong selection

The player must tap the matching word.

Core Game Loop
Start game.
Show target word.
Show two answer choices.
Player taps one choice.
Game checks if selected answer is the correct association.
If correct:
selected node glows
score increases
streak/combo may increase
next round appears quickly
If wrong:
selected node shows wrong marker
correct explanation appears
time penalty may apply
player taps to continue
Continue until the timer ends.
Show results/review screen.

Instruction

At the bottom:

or more generally:

For LexRush, I recommend:

That is clearer and supports nouns, verbs, adjectives, and phrases.

Interaction
Primary input

The player taps one of the two bottom answer nodes/cards.

Tap area should include:

the node circle
the answer label
nearby touch area

Do not make only the text tappable.

Correct Answer Behavior

When the player taps the correct option:

selected bottom node glows cyan/pink
line from target node to selected node lights up
small pulse animation appears
score increases
next round loads after a short delay

Possible feedback:

+100

or no text, just glow and score.

Wrong Answer Behavior

When the player taps the wrong option:

selected node shows an X or red highlight
correct answer may glow
explanation appears

Example from video:

Con means to persuade
through deception.

Then:

TAP TO CONTINUE

The game waits for the user to tap before moving on.

This is important because it turns wrong answers into learning moments.

Missed Behavior

If the player does not answer in time:

show Missed!
highlight the correct answer
optionally show the explanation
apply time penalty or simply continue

For LexRush, since your other games are timed, I recommend:

Missed: -2s after beginner rounds
Wrong: -3s
Correct: +100

But this can be tuned later.

Difficulty Progression
Beginner phase

Use obvious associations.

Examples:

dupe → trick
happy → joyful
small → tiny
angry → mad
clean → neat

Wrong options should be clearly unrelated.

Medium phase

Use closer distractors.

Examples:

attack → accuse / trick
fault → accuse / trick
mislead → trick / accuse

The user has to think more carefully.

Hard phase

Use nuanced word senses.

Examples:

charge → accuse / price
appeal → request / decorate
sound → healthy / noise
light → bright / not heavy

This is where the game becomes powerful because one word may have multiple meanings.

Beginner Ramp

For first-time players, use the first 5 rounds as teaching rounds:

only easy words
obvious correct association
slower timer pressure
no missed penalty
show clear instruction

Example first rounds:

happy → joyful / broken
fast → quick / dirty
small → tiny / loud
angry → mad / clean
cold → chilly / bright
Data Model

Use a structure like this:

class AssociationPrompt {
  final String targetWord;
  final String correctAnswer;
  final String wrongAnswer;
  final String explanation;
  final AssociationType type;
  final DifficultyTier difficulty;
  final bool beginnerSafe;

  const AssociationPrompt({
    required this.targetWord,
    required this.correctAnswer,
    required this.wrongAnswer,
    required this.explanation,
    required this.type,
    required this.difficulty,
    this.beginnerSafe = false,
  });
}

AssociationPrompt(
